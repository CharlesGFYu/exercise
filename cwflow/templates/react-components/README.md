```sh
  yarn
  or
  npm install
  or
  tnpm install
```

# 启动 server

```sh
  yarn start
  or
  tnpm run start
  or
  npm run start
```

# 动态打包

```sh
  yarn watch
  or
  tnpm run watch
  or
  npm run watch
```

# 构建压缩项目 js 文件

```sh
  yarn build
  or
  tnpm run build
  or
  npm run build
```

```sh
  tnpm run new component_name
```

```sh
  npm run compile

  tnpm run compile
```
