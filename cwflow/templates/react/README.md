# {{name}}

# 安装依赖包

```sh
  yarn
  or
  npm install
```

# 启动本地 server

```sh
  yarn start
  or 
  npm run start
```

# 动态打包

```sh
  yarn watch
  or
  npm run watch
```

# 构建压缩项目 js 文件

```sh
  yarn build
  or
  npm run build
```

# 项目发布

```sh
  只需发布 index.html 文件和 dist 目录
  /dist/
  index.html
```
